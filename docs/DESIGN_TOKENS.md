# 디자인 토큰 (색·역할)

`tailwind.config.js`에 정의된 색상과 용도 정리.

## 메인 액센트

| 토큰 | 용도 |
|------|------|
| `happy-*` | 메인 브랜드, CTA 버튼, 강조 |
| `happy-400` | 주요 버튼 배경 |
| `happy-700` | 버튼 텍스트, 링크 |

## 카드·배경

| 토큰 | 용도 |
|------|------|
| `cream-*` | 카드 배경, 부드러운 배경 |
| `lavender-*` | 헤더·섹션 구분 |
| `peach-*` | 작성 화면 상단 등 따뜻한 강조 |

## 위험·경고

| 토큰 | 용도 |
|------|------|
| `coral-*` | 에러, 삭제, 경고 |

## 보조 액센트

| 토큰 | 용도 |
|------|------|
| `mint-*` | 댓글·서브 요소 강조 |

## 다크 모드

- `dark:bg-stone-*`, `dark:text-stone-*` — 시스템 다크 모드 대응
- `dark:border-stone-*` — 테두리
